**현재 구현해 놓은 기능들**

- 검색
- TMDB Api 통신 완료, 영화 데이터 받아오기
- 영화 이미지 클릭시 영화 상세내용 모달창 팝업
- 모달창 내부 추천된 영화 이미지 클릭시 화면 전환

**앞으로 구현 할 기능들**

- 회원가입, 로그인, 로그인 JWT 토큰 인증
- 카카오톡 소셜 로그인
- 마이페이지
- 리스트에 영화 추가하기(찜목록)
- 해당 영화를 볼 수 있는 OTT 목록 리스트업
